Guide

A guide to use Elite BGS efficiently and completely

Systems and Factions Search

Search for any system or faction by going to the systems page or the factions page and start typing on the search bar. The table will be filtered dynamically as you search. You must start the search with the starting letters. So, searching for Qa' will give you a result. Searching for wakana won't give you any, although who knows if there is a system called Wakana.... In the results you can view the basic information of the system.

To see more details, click on the system or faction name.

In the details page, you can find the basic details as well as the states of all the factions in the system and the influence chart for the last 10 days. You can also sort and filter using the column headers. In the faction, states, one can see the trend of the pending and recovering states.

  • Red means that they are trending down
  • Blue means that they are not trening
  • Green means that they are trending up

One can predict the possibility of a state from going active among multiple pending states by seeing the trend.

In the graphs you can switch on or off a faction by clicking on the faction in the legend.

Log in

If you are in the home page.

If you are on some other page, click the cog icon and select Login

Logging in is only available through Discord OAuth, since most of the player community has a Discord server. It also helps in authorising an user for features as mentioned later.

Joining the Official Discord Server

Join the official Discord server by clicking on the Elite BGS Discord Server link in the About section.

You can join the server by clicking on the JOIN DISCORD SERVER button.

Logged in User Privileges

Although a user can get data for any system and faction without logging in, a logged in user has the benefit of having pinned factions and systems to monitor. These factions show up in the home page after logging in.

You can add more factions and systems to monitor using the + ADD button in the home page.

Or you can add by going to the systems page or the factions page, searching and selecting the system or faction and clicking on Monitor System or Monitor Faction.

You will need to monitor factions and systems seperately for each to show up.

All added systems and factions will be listed in your Profile. You can delete using the REMOVE button next to a system or faction.

A major feature of Elite BGS is that you can update system data manually. Elite BGS gets its data automatically through EDDN but that means one needs to actually make a jump into the system for the database to get updated. Now you can update the system data by seeing the System Map itself. Great for monitoring those fringe systems nobody visits.

To update the system data, you need to have access to the system edit functionality. This access is provided manually to representatives of factions who are present in that particular system. If you are a representative, join the Official Discord Server and request access. After due verification, access will be provided.

Once you get access, just go to the system view of your system. Press the EDIT button. to bring up a modal.

Editing a system involves updating all the factions data.

  • Influence must be in percentages value. Total sum of influence must add up to 100
  • Select a state from the drop down.
  • Turn on the switch if the faction is the one is control. Turning on any switch will turn of the rest. You cannot turn off a switch.
  • Remove any unneeded pending and recovering states by clicking on the tags
  • Selecting the pending and recovering states.
    • Select a state from the drop down.
    • Select a trend for the state.
    • Click on the button ADD PENDING/ADD RECOVERING

If a faction is not present in the system anymore, you can remove it by clicking the REMOVE FACTION button. You cannot add a new faction due to security and the only way to add is through an EDDN message.

Click the SAVE button to save your updates. If you make any mistake, you can press the RESET button to revert all your changes.

That's all folks. Happy Grinding!

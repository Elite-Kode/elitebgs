Hosted Bot

I am hosting the bot for all to use. Click the invite in the overview tab. You must have Manage Server permission for the server.

After inviting the bot, follow the steps in order:

  1. Run command @BGSBot myguild set to register your Discord guild(fancy term for a server). The command MUST be run by someone who has the Administrator permission in Discord.
  2. Next you need to add an admin role. An admin role is able to execute all commands. Find which role you want to add as an admin role. You can add many roles as admin roles. Next you need to find the id of the admin role(s). To do that prepend the mention of your role with \. Eg. if your role is admin, you need to use \@admin to see the id.
  3. With the role id, run the command @BGSBot adminroles add <id>. Replace <id> with the role found in the previous step. The command MUST be run by someone who has the Administrator permission in Discord. For eg. if Discord shows the id as <@&123456789012345678>, run the command as @BGSBot adminroles add 123456789012345678.
  4. Now, add a BGS Manager role. A BGS Manager would be able to execute commands related to BGS but not the bot's administrative commands. Run the command @BGSBot bgsrole set <id>. You need to have an admin role that you set in the previous step to execute this command. Also, there can be only 1 role for BGS Manager. Use the numeric id only like adminroles.
  5. Optionally, set up a BGS Channel using the command @BGSBot bgschannel set <id>. Replace <id> with the channel id that you want to convert to a BGS reporting channel. Use the numeric id only like adminroles.
  6. Optionally, set up forbidden roles. Users having these roles won't be able to execute any command even if they have admin roles or bgs manager roles. Run the command @BGSBot forbiddenroles add <id>. Use the numeric id only like adminroles. You can add as many roles as you like.
  7. Optionally, set up automated BGS Reporting. For this, a BGS Channel must be set up. To set up automated BGS Reporting, use a time in UTC with the following command @BGSBot bgsreport settime <time in UTC>. The time must be of the format HH:mm:ss. For eg. if the time to show the report is 15:25:32 UTC everyday, run the command as @BGSBot bgsreport settime 15:25:32.
  8. Optionally, set up sorting of all output. Setting this up will enable sorting of all data in the order of System or Faction name or influence in either. To set up sorting use the following command @BGSBot sort set <type> <order>. The type must be either influence or name and the order must be either increasing or decreasing. For eg. to sort according to influence in descending/decreasing order use the following command @BGSBot sort set influence decreasing

This concludes the set up of the bot.

Using the bot

The bot can be commanded to fetch data of systems and factions. The name is case insensitive but the entered name should reflect ingame name in English. The commands that the bot has currently are:

  1. @BGSBot factionstatus get <faction name in lower case>
    For eg. @BGSBot factionstatus get knights of karma
  2. @BGSBot systemstatus get <system name in lower case>
    For eg. @BGSBot systemstatus get qa'wakana

To create BGS Reports you need to monitor factions and systems. A faction or a system can be monitored as primary or not. Primary factions would be having detailed data while non primary ones would be having concise ones in a compact format. Only primary monitored systems is currently available in BGS reports.

To monitor a faction, command using:

  1. @BGSBot monitorfactions add <factionname>
    For eg. @BGSBot monitorfactions add knights of karma
  2. @BGSBot monitorfactions addprimary <factionname>
    For eg. @BGSBot monitorfactions addprimary knights of karma

To monitor a system, command using:

  1. @BGSBot monitorsystems add <systemname>
    For eg. @BGSBot monitorsystems add qa'wakana
  2. @BGSBot monitorsystems addprimary <systemname>
    For eg. @BGSBot monitorsystems addprimary qa'wakana

To generate a report type: @BGSBot bgsreport get

Administrative Commands

To remove or list any role or channel, use one of the following commands as suited:

  1. @BGSBot adminroles list
  2. @BGSBot adminroles remove <id>
    For eg. @BGSBot adminroles remove 123456789012345678
  3. @BGSBot bgsrole show
  4. @BGSBot bgsrole remove <id>
    For eg. @BGSBot bgsrole remove 123456789012345678
  5. @BGSBot bgschannel show
  6. @BGSBot bgschannel remove <id>
    For eg. @BGSBot bgschannel remove 123456789012345678
  7. @BGSBot forbiddenroles list
  8. @BGSBot forbiddenroles add <id>
    For eg. @BGSBot forbiddenroles remove 123456789012345678

To remove or list the monitored factions or systems:

  1. @BGSBot monitorfactions list
  2. @BGSBot monitorfactions remove <factionname>
    For eg. @BGSBot monitorfactions remove knights of karma
  3. @BGSBot monitorsystems list
  4. @BGSBot monitorsystems remove <systemname>
    For eg. @BGSBot monitorsystems remove qa'wakana

To unset or show the BGS Reporting time:

  1. @BGSBot bgsreport showtime
  2. @BGSBot bgsreport unsettime

To remove or show the sorting setup:

  1. @BGSBot sort show
  2. @BGSBot sort remove

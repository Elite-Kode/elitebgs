The complete RESTful API for EDDB

Currently EDDB is the greatest treasure of information of the galaxy and it is a user friendly tool for many pilots. But it lacks an API for developers. Elite BGS is an attempt to fill this gap by providing an API for EDDB data sources.

API

Access
Consumers must make a GET request at any endpoint.
Endpoints

Bodies

https://eddbapi.kodeblox.com/api/v1/bodies?<params>

Factions

https://eddbapi.kodeblox.com/api/v1/factions?<params>

Populated Systems

https://eddbapi.kodeblox.com/api/v1/populatedsystems?<params>

Stations

https://eddbapi.kodeblox.com/api/v1/stations?<params>

Systems

https://eddbapi.kodeblox.com/api/v1/systems?<params>
*Please note that the endpoints have been changed to their own subdomain and the version has been reset to 1
